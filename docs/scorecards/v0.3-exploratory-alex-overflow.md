## Purpose

This smoke pass checks whether the v0.3 Alex overflow redesign is coherent
before any n=30 scorecard:

- final task no longer repeats stale "Friday is risky" framing
- `honors_latest_intent` LLM judge distinguishes lexical mentions from genuine Friday commitment
- async judge can only downgrade
- cost/token logging works
- overflow difficulty tiers can be compared at N=0, N=20, and N=80

## Observations

- The v0.3 final-task redesign is coherent: every cell scored 30/30 task success and the LLM judge confirmed latest-intent credit.
- The high-overflow cell now shows a stable +5 point hybrid advantage over windowed at N=20 and N=80, coming from `uses_prior_outcome`.
- The remaining consistent miss is `scope_tradeoff`: drafts mention the post-launch fix window but not Tuesday as the fallback alternative.
- N=80 hybrid is wall-time expensive: 177 LLM calls and roughly 20 minutes for a single trial.

## Prereg Implication

Do not run n=30 until the prereg protocol is frozen and the team accepts the
wall-time/cost profile. Based on this smoke pass, N=80 hybrid costs about
$3.67 per trial and N=80 windowed costs about $1.22 per trial on this provider,
so an n=30 two-agent cell would be roughly $147 before retries or other agents.
